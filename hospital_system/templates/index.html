About

The Smart Patient Assistance and Staff Management System is a web-based hospital management solution designed to improve communication between patients and staff.

Patients can send requests for Water, Medicine, Nurse Assistance, or Emergency Help via web or physical ESP32 button panels. The system auto-prioritizes emergencies and tracks every request from start to finish.
